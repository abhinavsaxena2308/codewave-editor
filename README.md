# CodeWave - Multi-language Online Code Editor

CodeWave is a modern, web-based code editor that supports **HTML, CSS, JavaScript, and Python**. It features a responsive design, dark/light themes, live preview for HTML/CSS/JS, and an interactive Python environment powered by Skulpt.
